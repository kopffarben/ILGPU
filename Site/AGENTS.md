# Agent Context for Site

This directory hosts the Jekyll-based website for ILGPU. It contains markdown posts, layouts, and assets used to build the project homepage.

- `_posts/` – Blog posts and release announcements.
- `_layouts/` and `_includes/` – Shared HTML templates.
- `assets/` – CSS, images and JavaScript.
- `scripts/` – Helper scripts for site generation.
- `_config.yml` – Jekyll configuration.
