# Agent Context for Tools

Collection of helper utilities used during development.

- `CheckStyles/` – MSBuild targets enforcing code style rules.
- `CudaVersionUpdateTool/` – Console app that updates bundled CUDA libraries.
- `CopyrightUpdateTool/` – Script to sync copyright headers.
- `GenerateCompatibilitySuppressionFiles/` – Generates suppression lists for compatibility analyzers.
- `Directory.Build.props` – Shared project properties for these tools.
