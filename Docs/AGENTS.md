# Agent Context for Documentation

This directory contains end-user guides and tutorials.

- `01_Primers/` – Introduces GPU concepts and basic ILGPU usage.
- `02_Beginner/` – Step by step guides for new users.
- `03_Advanced/` – In depth articles about optimization and custom backends.
- `04_Upgrade-Guides/` – Information about migrating between ILGPU versions.
- `README.md` – Overview and links to all guides.
